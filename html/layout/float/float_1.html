float : 이미지 옆에 텍스트 배치하고 싶을 때 : 텍스트가 이미지를 피해가는 효과, 이미지를 따라 텍스트가 흐름

피해가지 않았으면 좋겠으면 하는 element에게 clear:both; 값을 주면 이미지를 따라 흐르지 않고 block 상태처럼 보이게 됨
단, both의 경우 무조건 양쪽 적용이지만 이미지에 적용된 float value 값과 clear value값이 동일해야 적용한다. 그러나 귀찮기 때문에 그냥 both를 사용한다.
